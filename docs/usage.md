# Usage

- Launch Noise Studio (installed or run `dist\NoiseStudio.exe`).
- Use the left settings panel to set resolution and live preview.
- Click `Generate Image Noise` to generate an image; images are automatically saved to `Documents/NoiseStudio/output/images`.
- Use `Play Sound` to generate and play audio; audio files are saved to `Documents/NoiseStudio/output/sound`.
- Use `Settings` menu to change sample rate and theme; settings are persisted to `Documents/NoiseStudio/config.json`.
